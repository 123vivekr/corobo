## Setup

1. Install the dependencies
   `pip install -r requirements.txt`

2. Set environment variables `COBOT_ROOT`, `COBOT_TOKEN` to the path of corobo
   repo and gitter token for cobot.

3. Run errbot: `cd COBOT_ROOT &&  errbot`
